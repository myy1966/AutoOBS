# AutoOBS

A simple GUI tool to automatically control the OBS Studio

## Requirements

### Python Package
| Package | Version tested |
| --- | --- |
| Python | 3.8 |
| toml | 0.10.2 |
| pynput | 1.7.4 |
| obs-websocket-py | 0.5.3 |
| PyQt5 | 5.15.5 |

### External Applications
| Application | Version tested |
| --- | --- |
| OBS Studio | 27.1.3 |
| obs-websocket | 4.9.1 |
